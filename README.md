# Partage

A simple file sharing web application with basic ACLs. (in french, *'partage'*
means *'sharing'*)

This project is **work in progress**

## Installation

- **From npm** (not yet published)
  ```
$ npm install partage
```

- **From git**
  ```
$ git clone https://github.com/vdust/partage.git
```
